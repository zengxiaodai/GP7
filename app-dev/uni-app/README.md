# uniapp简介

- uniapp：跨平台技术（一套前端代码，打包成多个平台的应用）
- uniapp：各种小程序、android app、ios app、快应用、webapp(h5)等等。
- 现在呈现的市场趋势是：很多中小型公司用uniapp做android app。（uniapp做小程序的公司比较少）

- 2018年9月，uniapp开源了（发布了）。

- 开发特点：一种是vue2风格、另一种是vue3风格（2021年8月左右）。
